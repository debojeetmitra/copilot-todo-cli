# Copilot Todo CLI

A simple command-line Todo List application built with **Node.js** using **GitHub Copilot**. This project was created as part of the **MLH Global Hack Week: Season Launch** challenge to explore AI-assisted development.

## ✨ Features

- ➕ Add new tasks
- 📋 View all tasks
- ❌ Delete tasks
- 🚪 Exit the application
- 🧠 Built with the help of GitHub Copilot

## 🛠️ Technologies Used

- Node.js
- JavaScript
- GitHub Copilot
- GitHub Codespaces
- Git

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/debojeetmitra/copilot-todo-cli.git
cd copilot-todo-cli
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
node index.js
```

Example menu:

```text
===== Todo CLI =====

1. Add Task
2. View Tasks
3. Delete Task
4. Exit

Choose an option:
```

## 🎯 Purpose

This project was created to practice using **GitHub Copilot** as an AI coding assistant for building a simple application and to gain hands-on experience with GitHub Codespaces.
